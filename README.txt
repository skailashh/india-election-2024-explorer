A React-based web app built with Vite to explore and visualize election data from CSV files.
Uses PapaParse for fast client-side CSV parsing, Lucide React for clean icon components, and Vite as a lightning-fast dev server and build tool.
Core logic handles file input, data parsing, and dynamic rendering of insights through React components.
To run locally: install Node.js, run npm install, then npm run dev and open the displayed localhost URL.
